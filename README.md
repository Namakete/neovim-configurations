### ☄️ About

These are custom configurations based on [nvchad](https://github.com/NvChad/NvChad).

### 🚀 Setup

```bash
git clone https://github.com/Namakete/neovim-configurations.git ~/.config/nvim --depth 1 && nvim
```
### 🗑️ Uninstall

```bash
rm -rf ~/.config/nvim
rm -rf ~/.local/share/nvim
rm -rf ~/.cache/nvim
```
